# DC 悬浮记事本 - Electron 版

## 功能特性

- 📝 **快速记录**：文字、截图、语音、剪贴板一键录入
- 🏷️ **智能分类**：内置关键词分类（开箱即用），可选 AI 智能分类
- 🔍 **快速找回**：全文搜索 + 分类筛选 + 时间线浏览
- 🎨 **精美 UI**：毛玻璃风格，彩色分类标签
- ⌨️ **全局快捷键**：快速唤起/隐藏
- 🔒 **本地存储**：数据保存在本地，隐私安全
- 📌 **始终置顶**：窗口悬浮在所有应用之上
- 💬 **DC 集成**：可发送笔记为 D-Chat 待办

## 安装与运行

### 开发模式

```bash
cd ~/dchat-workplace/dc-notepad/electron
npm install
npm start
```

### 打包应用

```bash
# Mac
npm run build:mac

# Windows
npm run build:win

# Linux
npm run build:linux

# 全部平台
npm run build:all
```

生成的安装包在 `dist/` 目录下。

## 使用方法

### 全局快捷键

| 平台 | 快捷键 | 说明 |
|------|--------|------|
| Mac | `Alt + N` | 唤起/隐藏记事本 |
| Windows/Linux | `Ctrl + Shift + N` | 唤起/隐藏记事本 |

### 基本操作

| 操作 | 说明 |
|------|------|
| 输入文字 + `Enter` | 保存为笔记 |
| `Cmd/Ctrl + V` | 粘贴截图 |
| 🎤 按钮 | 语音输入（转文字） |
| 📋 按钮 | 读取剪贴板内容 |
| 分类标签 | 点击筛选不同类型笔记 |
| 搜索框 | 全文搜索笔记内容 |
| 托盘图标 | 点击显示/隐藏，右键菜单退出 |

### 分类说明

默认内置 6 种分类，开箱即用：

| 分类 | 图标 | 关键词示例 |
|------|------|-----------|
| 待办 | ✅ | 要做、完成、跟进、记得、TODO |
| 临时信息 | 🔑 | 密码、token、API key、地址 |
| 会议 | 📋 | 会议、讨论、评审、周会、对齐 |
| 灵感 | 💡 | 想法、试试、可以考虑、优化 |
| 疑问 | ❓ | 为什么、怎么、不懂、请问 |
| 其他 | 📌 | 其他内容 |

### AI 分类（可选）

如需更智能的分类效果，可启用 AI：

1. 点击「设置」→「🤖 AI 分类设置」
2. 勾选「启用 AI 智能分类」
3. 填写 LiteLLM 服务信息：
   - 服务地址（如：`127.0.0.1`）
   - 端口（如：`9999`）
   - API Key
   - 模型名称（如：`glm-5-external`）
4. 点击「测试」验证连接
5. 保存配置

> 提示：AI 分类为可选功能，不配置也能正常使用本地关键词分类。

### 发送到 D-Chat

可将笔记创建为 DC 待办：

1. 点击笔记右上角的「同步」按钮
2. 首次使用需配置 Smartwork 凭证
3. 自动创建待办并设置明天为截止日期

## 数据存储位置

| 平台 | 数据路径 |
|------|----------|
| Mac | `~/Library/Application Support/dc-notepad/` |
| Windows | `%APPDATA%/dc-notepad/` |
| Linux | `~/.config/dc-notepad/` |

## 注意事项

- 首次运行需要 `npm install` 安装依赖
- 打包后的应用约 100-150MB（包含 Electron 运行时）
- 数据文件为本地 JSON，可手动备份
